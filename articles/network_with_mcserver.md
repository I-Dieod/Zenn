# はじめに
この記事はネットワークインフラ知識ゼロの人間が、Minecraftのマルチプレイ用サーバーを建てるためにGeminiに教えてもらいながら色々やった結果をまとめたものです。
インフラエンジニア入門みたいな知識の寄せ集めになります。

## 対象読者
- Minecraftのマルチプレイサーバーを建ててみたい人

# 背景
最近Minecraftのマルチプレイにハマった影響で、「自分もマルチプレイ用サーバーを建ててホスティングしてみたい！」「PC一台でマルチプレイサーバーとクライアントの両立ができないかな？」と思い、サーバーの建て方について手を出してみました。
